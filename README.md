# Training Planner (План тренировок)

**Автор:** Высокос Анастасия

## Описание

Training Planner — это графическое приложение для планирования тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности, просматривать список тренировок, фильтровать по типу или дате, удалять записи. Данные сохраняются в JSON-файл и загружаются при запуске.

## Как запустить приложение

1. Убедитесь, что установлен Python 3.6 или выше.
2. Скачайте или склонируйте репозиторий:
   ```bash
   git clone https://github.com/parklir/Training-Planner1.git
   cd training-planner1

3. Запустите приложение:
   ```bash
   python main.py
