# Random Task Generator

**Автор:** Высокос Анастасия

## Описание

Random Task Generator — это графическое приложение для генерации случайных задач. Позволяет создавать список доступных задач, генерировать случайную задачу с указанием типа, сохранять историю сгенерированных задач и фильтровать её по типу.

## Как запустить приложение

1. Убедитесь, что установлен Python 3.6 или выше.
2. Скачайте или склонируйте репозиторий:
   ```bash
   git clone https://github.com/parklir/Random-Task-Generator.git
   cd random-task-generator
3. Запустите приложение:
   ```bash
   python main.py
   ```
